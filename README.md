## Accounts

The following API allows to process all standard CRUD operations with Accounts.

### Create account

Request:
```http
POST /api/accounts/create
Content-type: application/json

{
    "login": "dstreltsov",
    "password": "example-password",
    "email": "dstreltsov@test.zz",
    "birthYear": 1999
}
```
Response:
```http
{

}
```
